# Hotkey
几个有用但是用的不多的快捷键

### 1.合并行和文本（Join Lines and Literals）

  * 描述：这个操作比起在行末使劲按删除键爽多了！该操作遵守格式化规则，同时：
    * 合并两行注释，同时移除多余的//；
    * 合并多行字符串，移除+和双引号；
    * 合并字段的声明和初始化赋值；
  * 快捷键：Ctrl + Shift + J；
